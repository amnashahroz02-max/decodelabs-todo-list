# To-Do List App 📝

A command-line To-Do List application built with Python.
Developed as Project 1 for DecodeLabs Internship 2026.

## Features
- Add tasks to your list
- View all tasks in numbered format
- Simple and clean menu interface

## How to Run
```bash
python todo.py
```

- Functions and how to define them
- enumerate() for professional list display
- while loops for menu systems
- if/elif/else logic

## Tech Stack
- Python 3.14
